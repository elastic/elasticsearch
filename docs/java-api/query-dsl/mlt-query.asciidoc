==== More Like This Query (mlt)

See:
 * {ref}/query-dsl-mlt-query.html[More Like This Query]

[source,java]
--------------------------------------------------
QueryBuilder qb = moreLikeThisQuery("name.first", "name.last") <1>
    .like("text like this one")                                <2>
    .minTermFreq(1)                                            <3>
    .maxQueryTerms(12);                                        <4>
--------------------------------------------------
<1> fields
<2> text
<3> ignore threshold
<4> max num of Terms in generated queries

[[esql-functions]]
== ESQL functions

++++
<titleabbrev>Functions</titleabbrev>
++++
:keywords: {es}, ESQL, {es} query language, functions
:description: ESQL supports various functions for calculating values.

<<esql-row,`ROW`>>, <<esql-eval,`EVAL`>> and <<esql-where,`WHERE`>> support
these functions:

* <<esql-abs>>
* <<esql-auto_bucket>>
* <<esql-case>>
* <<esql-cidr_match>>
* <<esql-concat>>
* <<esql-date_format>>
* <<esql-date_trunc>>
* <<esql-e>>
* <<esql-is_finite>>
* <<esql-is_infinite>>
* <<esql-is_nan>>
* <<esql-is_null>>
* <<esql-length>>
* <<esql-mv_avg>>
* <<esql-mv_concat>>
* <<esql-mv_count>>
* <<esql-mv_dedupe>>
* <<esql-mv_max>>
* <<esql-mv_median>>
* <<esql-mv_min>>
* <<esql-mv_sum>>
* <<esql-now>>
* <<esql-pow>>
* <<esql-round>>
* <<esql-split>>
* <<esql-starts_with>>
* <<esql-substring>>

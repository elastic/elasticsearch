[[esql-abs]]
=== `ABS`
[.text-center]
image::esql/functions/signature/abs.svg[Embedded,opts=inline]

Returns the absolute value.

[source,esql]
----
FROM employees
| KEEP first_name, last_name, height
| EVAL abs_height = ABS(0.0 - height)
----

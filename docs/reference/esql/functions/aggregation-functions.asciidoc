==== {esql} aggregate functions

++++
<titleabbrev>Aggregate functions</titleabbrev>
++++

* <<esql-min>>
* <<esql-percentile>>

* <<esql-std_dev>>
* <<esql-sum>>
* <<esql-top>>
* <<esql-values>>
* <<esql-weighted_avg>>

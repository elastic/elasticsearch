### DATE_TRUNC
Rounds down a date to the closest interval since epoch, which starts at `0001-01-01T00:00:00Z`.

```
FROM employees
| KEEP first_name, last_name, hire_date
| EVAL year_hired = DATE_TRUNC(1 year, hire_date)
```

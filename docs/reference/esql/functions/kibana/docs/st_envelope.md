### ST_ENVELOPE
Determines the minimum bounding box of the supplied geometry.

```
FROM airport_city_boundaries
| WHERE abbrev == "CPH"
| EVAL envelope = ST_ENVELOPE(city_boundary)
| KEEP abbrev, airport, envelope
```

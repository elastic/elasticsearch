[[esql-mv_avg]]
=== `MV_AVG`

*Syntax*

[source,esql]
----
MV_AVG(expression)
----

*Parameters*

`expression`::
Multivalue expression.

*Description*

Converts a multivalued expression into a single valued column containing the
average of all of the values.

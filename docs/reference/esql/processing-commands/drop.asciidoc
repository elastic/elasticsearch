[[esql-drop]]
=== `DROP`

**Syntax**

[source,esql]
----
DROP columns
----

*Parameters*

`columns`::
A comma-separated list of columns to remove. Supports wildcards.

*Description*

The `DROP` processing command removes one or more columns.

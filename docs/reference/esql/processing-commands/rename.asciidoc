[[esql-rename]]
=== `RENAME`

Use `RENAME` to rename a column using the following syntax:

[source,esql]
----
RENAME <new-name> = <old-name>
----

----
include::{esql-specs}/docs.csv-spec[tag=rename]
----

If a column with the new name already exists, it will be replaced by the new
column.

[[esql-where]]
=== `WHERE`

**Syntax**

[source,esql]
----
WHERE expression
----

*Parameters*

`expression`::
A boolean expression.

*Description*

The `WHERE` processing command produces a table that contains all the rows from
the input table for which the provided condition evaluates to `true`.

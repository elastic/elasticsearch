[[indices-get-template]]
=== Get index template API
++++
<titleabbrev>Get template index</titleabbrev>
++++

Returns information about one or more index templates.

////
[source,console]
--------------------------------------------------
PUT _template/template_1
{
 "index_patterns" : ["te*"],
  "settings": {
    "number_of_shards": 1
  }
}
--------------------------------------------------
// TESTSETUP
////

[source,console]
--------------------------------------------------
GET /_template/template_1
--------------------------------------------------

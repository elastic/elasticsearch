## `SHA256` [esql-sha256]

**Syntax**

:::{image} ../../../../../images/sha256.svg
:alt: Embedded
:class: text-center
:::

**Parameters**

`input`
:   Input to hash.

**Description**

Computes the SHA256 hash of the input.

**Supported types**

| input | result |
| --- | --- |
| keyword | keyword |
| text | keyword |

**Example**

```esql
FROM sample_data
| WHERE message != "Connection error"
| EVAL sha256 = sha256(message)
| KEEP message, sha256;
```

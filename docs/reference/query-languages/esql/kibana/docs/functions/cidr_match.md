### CIDR_MATCH
Returns true if the provided IP is contained in one of the provided CIDR blocks.

```
FROM hosts 
| WHERE CIDR_MATCH(ip1, "127.0.0.2/32", "127.0.0.3/32") 
| KEEP card, host, ip0, ip1
```

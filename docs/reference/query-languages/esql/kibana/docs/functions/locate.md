### LOCATE
Returns an integer that indicates the position of a keyword substring within another string.
Returns `0` if the substring cannot be found.
Note that string positions start from `1`.

```esql
ROW a = "hello"
| EVAL a_ll = LOCATE(a, "ll")
```

### QSTR
Performs a [query string query](/reference/query-languages/query-dsl/query-dsl-query-string-query.md). Returns true if the provided query string matches the row.

```
FROM books
| WHERE QSTR("author: Faulkner")
| KEEP book_no, author
| SORT book_no
| LIMIT 5
```

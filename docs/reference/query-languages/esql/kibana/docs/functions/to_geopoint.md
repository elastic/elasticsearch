### TO_GEOPOINT
Converts an input value to a `geo_point` value.
A string will only be successfully converted if it respects WKT Point format.

```
ROW wkt = "POINT(42.97109630194 14.7552534413725)"
| EVAL pt = TO_GEOPOINT(wkt)
```

### TO_GEOSHAPE
Converts an input value to a `geo_shape` value.
A string will only be successfully converted if it respects WKT format.

```
ROW wkt = "POLYGON ((30 10, 40 40, 20 40, 10 20, 30 10))"
| EVAL geom = TO_GEOSHAPE(wkt)
```

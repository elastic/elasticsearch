# SHOW CATALOGS [sql-syntax-show-catalogs]

```sql
SHOW CATALOGS
```

**Description**: List the available catalogs and their types.

```sql
SHOW CATALOGS;

     name         |     type
------------------+---------------
javaRestTest         |local
my_remote_cluster |remote
```
